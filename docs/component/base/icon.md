## 基础用法

基础的按钮用法。

直接通过设置类名为 **name** 属性 来使用即可。例如：

<template>
  <div class="demo-icon">
    <p-icon name="edit"></p-icon>
    <p-icon name="home"></p-icon>
    <p-icon name="share"></p-icon>
    <p-icon name="delete"></p-icon>
  </div>
</template>


### 使用
```html
    <p-icon name="edit"></p-icon>
    <p-icon name="home"></p-icon>
    <p-icon name="share"></p-icon>
    <p-icon name="delete"></p-icon>
```
